About This Project

Digital Image Processing

This project demonstrates real image processing techniques using OpenCV and NumPy, not just CSS filters. Each era uses specific transformations like sepia matrices, histogram equalization, bilateral filters, and color space conversions.

Technologies Used

  • Backend: Python Flask
  • Image Processing: OpenCV, Pillow, NumPy
  • Frontend: HTML5, CSS3, JavaScript
  • Techniques: Color space transforms, CLAHE, bilateral filtering, convolution
